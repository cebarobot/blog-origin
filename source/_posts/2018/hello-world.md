## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Run server

``` bash
$ hexo server
$ hexo s
```

### Generate static files

``` bash
$ hexo generate
$ hexo g
```

### Deploy to remote sites

``` bash
$ hexo deploy
$ hexo d
```
